Available Actions

Continuous Dictation

Just press the microphone and start talking. You should be able to see the text result into the main textarea. Try to write a letter with your voice!

Search Inside Document

Once the microphone is active, try the following voice commands:

English
Voice Command Action
"Hello Search" Enables to activate search input keyword
"finish hello search" Finishes hello search
Español
Comando de Voz Acción
"hola buscar" Habilita la Estrategia para el cambio de Título
"terminar hola buscar" Finaliza la Estrategia para el cambio de Título
